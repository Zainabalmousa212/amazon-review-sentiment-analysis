# amazon-review-sentiment-analysis

# 📊 # Sentiment Analyzer for Amazon Reviews

## 📌 Overview

This project implements a **Sentiment Analysis system** using Python to analyze Amazon customer reviews for the **Samsung Galaxy S4**.

The system processes review text and automatically classifies it into three sentiment categories:

* 😊 Positive
* 😐 Neutral
* 😞 Negative

The classification is based on **TextBlob polarity scores**, which measure the sentiment orientation of each review.

---

## ⚙️ Technologies Used

* 🐍 Python
* 📊 Pandas
* 🧠 TextBlob
* 📉 Matplotlib
* 📓 Jupyter Notebook

---

## 📂 Dataset

The dataset contains Amazon reviews for mobile phones.

For this project:

* Reviews related to **Samsung Galaxy S4** were filtered.
* Total analyzed reviews: **6056 reviews**

---

## 📈 Analysis Output

The system produces:

* Sentiment classification for each review
* Polarity score for review text
* Sentiment distribution visualization
* Polarity histogram

These insights help summarize **customer satisfaction and opinions** about the product.
